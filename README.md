# Blood Donation Finder

A modern, responsive web application for connecting blood donors with those in need. Built with React and Tailwind CSS for a beautiful and user-friendly experience.

## 🩸 About the Project

Blood Donation Finder is a frontend-only web application that helps people find blood donors in emergency situations. The application features a clean, modern interface with intuitive navigation and real-time search capabilities.

## ✨ Features

- **🏠 Home Page**: Welcome page with project introduction and navigation
- **📝 Donor Registration**: Easy-to-use form for blood donors to register
- **🔍 Find Donors**: Search and filter donors by blood type and location
- **📱 Responsive Design**: Works perfectly on all devices (mobile, tablet, desktop)
- **🎨 Modern UI**: Beautiful gradients, animations, and glass morphism effects
- **⚡ Fast Performance**: Built with Vite for optimal loading speeds

## 🛠️ Technology Stack

- **Frontend Framework**: React.js (v18.2.0)
- **Build Tool**: Vite
- **Styling**: Tailwind CSS (v3.3.5)
- **Routing**: React Router (v6.20.0)
- **Icons**: Emoji icons for better visual appeal

## 🚀 Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository** (if not already done):
   ```bash
   git clone https://github.com/nanthakumar-aids/WEBSITE-BUILDING-PROJECT.git
   cd WEBSITE-BUILDING-PROJECT
   ```

2. **Navigate to the frontend directory**:
   ```bash
   cd frontend
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

5. **Open your browser** and visit `http://localhost:5173`

## 📱 Usage

### For Blood Donors:
1. Click "Register as Donor" from the home page
2. Fill out the registration form with your details
3. Submit the form to register (demo mode - data is stored locally)

### For Those Seeking Blood:
1. Click "Find Blood Donor" from the home page
2. Select your required blood type from the dropdown
3. Enter your city/location
4. View matching donors and contact them directly

## 🎨 Design Features

- **Gradient Backgrounds**: Beautiful red and blue gradients representing blood donation
- **Glass Morphism**: Modern card designs with backdrop blur effects
- **Smooth Animations**: Hover effects and transitions for better user experience
- **Color-Coded Blood Types**: Different colors for each blood group (A+, A-, B+, B-, AB+, AB-, O+, O-)
- **Responsive Layout**: Mobile-first design that adapts to all screen sizes
